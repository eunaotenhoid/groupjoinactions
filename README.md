# Group Join Actions

Group Join Actions allows administrators to automate rewards, notifications, and visual styling when a user joins a specific group. It is a powerful tool for managing VIP groups, staff recruitment, or community milestones by providing points, invites, items, and professional visual indicators automatically.

## Main Features

- **Auto Points Reward**: Automatically adds points to the user's account upon joining a group (Supports Ultimate Points and APS).
- **Auto Invites Reward**: Automatically adds invites to the user's account (Supports Invitation Extension).
- **UltimateShop Integration**: Automatically grants a specific item from UltimateShop to the user upon joining a group.
- **Custom Welcome PM**: Sends a customized Private Message to the new group member with variable support (`{USERNAME}`, `{GROUP_NAME}`, etc.).
- **Post Border Styling**: Dynamically adds a colored border to user posts based on their primary group's color.
- **Group Icons**: Displays professional icons next to the username to visually distinguish special group members.

## Installation

1. Download the extension.
2. Copy the whole archive content to `/ext/eunaumtenhoid/groupjoinactions`.
3. Go to your phpBB board > Administration Control Panel > Customise > Manage extensions.
4. Locate **Group Join Actions** and click **Enable**.

## Update Instructions

1. Go to your phpBB board > Administration Control Panel > Customise > Manage extensions.
2. Locate **Group Join Actions** and click **Disable**.
3. Delete all files of the extension from `/ext/eunaumtenhoid/groupjoinactions`.
4. Upload all the new files to the same location.
5. Go back to the ACP and click **Enable** on the extension.
6. Purge the board cache.
